<h3>CNH API</h3>

Projeto consiste em uma API de CRUD de Usuários do aplicativo de CNH. Conectando Instrutores de direção e Alunos

O sistema possui doi tipos de Usuários
- Instrutor 
- Aluno

> Dados Coletados dos Alunos
 - Nome
 - Email
 - CPF
 - Telefone
 - Foto
 - País
 - Estado
 - Cidade
 - Bairro

> Dados Coletados dos Instrutores
 - Nome
 - Email
 - CPF
 - Telefone
 - Foto
 - País
 - Estado
 - Cidade
 - Bairro

>Usuário
- Email
- Senha
- Tipo (1 - Instrutor, 2 - Aluno)
